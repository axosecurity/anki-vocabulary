# Anki Vocabulary Flashcards

System prompts and data for generating Anki flashcards for English-to-Bengali vocabulary learning.

## Contents

- `prompt-vocabulary.md` — System prompt for general vocabulary card generation (with synonym swap, extra cards, verb conjugation)
- `prompt-preposition.md` — System prompt for preposition pattern flashcard generation
- `voca1_anki.txt` — Anki-importable vocabulary cards
- `voca1.txt` — Source vocabulary list
- `voca5.txt` — Additional vocabulary entries
